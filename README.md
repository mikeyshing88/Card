Card

## <a name="installing">Installing</a>
### Mac / Linux / Windows
```bash
npm cache clean
npm install
```
Or
```bash
yarn
yarn start
```

## <a name="launching">Launching</a>
| Task                | Usage                                                        |
| ------------------- | ------------------------------------------------------------ |
| `npm run start` or `yarn start`     | Opens local server |
| `npm run build` or `yarn build`     | Builds production |
